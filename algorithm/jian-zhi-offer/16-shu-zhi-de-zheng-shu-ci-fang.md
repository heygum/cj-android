# 16 数值的整数次方

![https://leetcode-cn.com/problems/shu-zhi-de-zheng-shu-ci-fang-lcof/](../../.gitbook/assets/image%20%2813%29.png)

大佬的解题:

```text
class Solution {
    public double myPow(double x, int n) {
        if(x == 0) return 0;
        long b = n;
        double res = 1.0;
        if(b < 0) {
            x = 1 / x;
            b = -b;
        }
        while(b > 0) {
            if((b & 1) == 1) res *= x;
            x *= x;
            b >>= 1;
        }
        return res;
    }
}
```
