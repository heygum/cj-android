# 32 - II. 从上到下打印二叉树 II



![https://leetcode-cn.com/problems/cong-shang-dao-xia-da-yin-er-cha-shu-ii-lcof/](../../.gitbook/assets/tu-pian-%20%287%29.png)

Noob's Solution:

```text
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> big = new ArrayList<>();
            if(root == null)
                return big;
           Queue<TreeNode> queue = new LinkedList<>();
           queue.add(root);
           List<Integer> small = new ArrayList<>();
           int change = 1;
           while(!queue.isEmpty())
           {
               TreeNode now = queue.poll();
               small.add(now.val);
               change--;
               if(change == 0)
               {
                   big.add(small);
                   small = new ArrayList<Integer>();
               }
               if(now.left != null)
                    queue.add(now.left);
                if(now.right != null)
                    queue.add(now.right);
                if(change == 0)
                    change = queue.size();
           }
           return big;
    }
}
```
