# 47. 礼物的最大价值

![](../../.gitbook/assets/image%20%2837%29.png)

```text
class Solution {
    public int maxValue(int[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        for(int y = 0; y < n; y++){
            for(int x = 0; x < m; x++){
                if(x == 0 && y == 0) 
                    continue;
                if(y == 0) 
                    grid[0][x] += grid[0][x-1];
                else if(x == 0) 
                    grid[y][0] += grid[y-1][0];
                else
                    grid[y][x] += Math.max(grid[y-1][x],grid[y][x-1]);
            }
        }
        return grid[n-1][m-1];
    }
}
```

**注意 int \[ 列 \]\[ 行 \].**
