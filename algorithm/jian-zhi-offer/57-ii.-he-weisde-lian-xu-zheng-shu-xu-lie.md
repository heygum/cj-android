# 57 - II. 和为s的连续正数序列

![https://leetcode-cn.com/problems/he-wei-sde-lian-xu-zheng-shu-xu-lie-lcof/](../../.gitbook/assets/tu-pian-%20%2826%29.png)

My code \( 数字一大 就输出 \[\] \):

```text
class Solution {
    public int[][] findContinuousSequence(int target) {
        int[] nums = new int[target];
        List<int[]> node = new ArrayList<>();
        int l = 0, r = target - 1 ;
        for(int i = 0; i < nums.length;i++)
            nums[i] = i + 1; 
        while(l != r){
            if(r > nums.length -1)
                break;
            int total = 0;
            for(int i = l; i <= r;i++)
                total += nums[i];
            if(total > target)
                r--;
            else if(total < target){
                r++;
                l++;
            }
            else if(total == target){
                int[] out = new int[r - l + 1];
                for(int i = 0; i < (r - l + 1); i++)
                    out[i] = nums[i+l];
                node.add(out);
                r++;
                l++;
            } 
        }
        return node.toArray(new int[node.size()][]);
    }
}
```
