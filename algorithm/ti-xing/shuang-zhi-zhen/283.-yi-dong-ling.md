# 283. 移动零

![](../../../.gitbook/assets/tu-pian-%20%2853%29.png)

My code:

```text
class Solution {
    public void moveZeroes(int[] nums) {
        int l = 0;
        for(int i = 0;i < nums.length;i++){
            if(nums[i] != 0)
                nums[l++] = nums[i];
        }
        for(int i = l;i < nums.length;i++)
            nums[i] = 0;
    }
}
```
