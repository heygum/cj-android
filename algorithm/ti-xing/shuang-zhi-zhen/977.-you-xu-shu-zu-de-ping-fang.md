# 977. 有序数组的平方

![](../../../.gitbook/assets/tu-pian-%20%2849%29.png)

```text
class Solution {
    public int[] sortedSquares(int[] nums) {
        int r = nums.length -1;
        int l = 0;
        int[] out = new int[nums.length];
        for(int i = nums.length -1 ;i >= 0 ;i--){
            if(nums[r] * nums[r] > nums[l] * nums[l]){
                out[i] = nums[r] * nums[r];
                r--;
            } else {
                out[i] = nums[l] * nums[l];
                l++;
            }
        }
        return out;
    }
}
```
