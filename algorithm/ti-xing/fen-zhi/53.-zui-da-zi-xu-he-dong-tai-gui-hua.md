# 53. 最大子序和\(动态规划\)

![](../../../.gitbook/assets/tu-pian-%20%2854%29.png)

```text
class Solution {
    public int maxSubArray(int[] nums) {
        int out = nums[0];
        int pre = 0;
        for(int num: nums){
            pre = Math.max(pre+num,num);
            out = Math.max(out,pre);
        }

        return out;
    }
}
```
