My code:

```text
/* The isBadVersion API is defined in the parent class VersionControl.
      boolean isBadVersion(int version); */

public class Solution extends VersionControl {
    public int firstBadVersion(int n) {
        int first = -1;
        int l = 1, r = n;
        int mid = l+ (r-l)/2;
        while(mid >= l && mid <= r){
            if(isBadVersion(mid) == true){
                first = mid;
                r = mid-1;
            }
            else l = mid+1;
            mid= l+ (r-l)/2;
        }
        return first;
    }
}



//要注意的是 int溢出问题
// l+ (r-l)/2 或
// l+ (r-l)>>1 或
// (l&r) + ((l^r)>>1)
```
