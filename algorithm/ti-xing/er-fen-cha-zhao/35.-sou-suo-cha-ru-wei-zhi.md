# 35. 搜索插入位置

![](../../../.gitbook/assets/tu-pian-%20%2847%29.png)

My code:

```text
class Solution {
    public int searchInsert(int[] nums, int target) {
        int l = 0;
        int r = nums.length -1 ;
        int mid = (l&r) + ((l^r)>>1);
        while(l <= r){
            mid = (l&r) + ((l^r)>>1);
            if(nums[mid] == target) return mid;
            if(nums[mid] < target){
                l = mid+1;
            }
            else r = mid-1;
        }
        return target > nums[mid]?mid+1:mid;
    }
}
```
